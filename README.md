
تنفيذ بسيط لـ المخزن الدائري (Circular Buffer) بلغة C باستخدام مبدأ FIFO.

آلية العمل:

- تخزين الأحرف في مصفوفة بحجم ثابت (60)
- استخدام "head" للقراءة و "tail" للكتابة و "count" للعدد
- الالتفاف باستخدام:

(index + 1) % SIZE

الخطوات:

1. إدخال الاسم
2. إضافة ""ESY-CE""
3. تخزين الأحرف في المخزن
4. قراءة وطباعة الناتج

مثال:

الإدخال: Ali
الناتج: AliESY-CE
